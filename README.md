# 🔍 Network Troubleshooting Tool

**Production-level CLI tool for diagnosing real-world network issues using TCP/IP protocols.**

---

## 🚀 Features

* ✅ Ping Test (ICMP)
* ✅ DNS Resolution
* ✅ Host/IP Validation
* ✅ TCP Port Scanning (80, 443, etc.)
* ✅ Traceroute Analysis
* ✅ Multithreading for faster execution

---

## 🧠 Real-World Insight

> Some servers block ICMP (ping), so a host may appear **DOWN** even if it is reachable.
> This tool uses **TCP port checks** to confirm actual service availability.

---

## 🖥️ Usage

```bash
python network_tool.py google.com github.com --ports 80 443
```
